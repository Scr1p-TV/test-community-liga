# UFL Live Draft System

## Features
- Live Draft Board
- Echtzeit Updates mit Firebase
- GitHub Pages Hosting
- Automatischer Timer
- Zuschauerfreundlich
- Kostenlos

---

# SCHRITT 1 — GitHub Repository

1. Gehe auf github.com
2. Neues Repository erstellen
3. Dateien hochladen

---

# SCHRITT 2 — Firebase

1. Gehe auf firebase.google.com
2. Neues Projekt erstellen
3. Realtime Database aktivieren
4. Testmodus aktivieren

---

# SCHRITT 3 — Web App erstellen

1. Firebase Einstellungen öffnen
2. Web App hinzufügen
3. Config kopieren
4. In firebase-config.js einfügen

---

# SCHRITT 4 — GitHub Pages aktivieren

1. Repository öffnen
2. Settings
3. Pages
4. Branch = main
5. Save

Danach bekommst du:
https://DEINNAME.github.io/REPOSITORY

---

# SCHRITT 5 — Live Draft starten

- Website öffnen
- Spieler draften
- Zuschauer sehen alles live
